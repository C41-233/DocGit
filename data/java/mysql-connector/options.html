URL格式：jdbc:mysql://<ip>:<port>/<database>

参数 类型 描述 默认值
autoReconnect boolean 自动重连 false
autoReconnectForPools boolean 针对数据库连接池的重连策略 false
capitalizeTypeNames boolean 数据定义名称以大写表示
characterEncoding string 字符集
connectTimeout int 建立连接的超时时间（毫秒） 0
failOverReadOnly boolean 自动重连后设置为只读 true
initialTimeout int 尝试重连时间间隔（秒） 2
interactiveClient boolean 设置CLIENT_INTERACTIVE标志，根据INTERACTIVE_TIMEOUT而不是WAIT_TIMEOUT向MySQL通报超时连接。 false
maxReconnect boolean 尝试重连次数 3
maxRows int 结果集最大行数
password string 密码
relaxAutocommit boolean 自动提交
socketFactory string 驱动程序用于创建与服务器套接字连接的类的名称，该类必须实现了接口com.mysql.jdbc.SocketFactory com.mysql.jdbc.StandardSocketFactory
socketTimeout int 套接字读写超时时间（毫秒） 0
useConfigs boolean 使用配置属性列表 false
useUnicode boolean UNICODE编码 false
user string 用户名
